# Chandan Adhikari - Personal Portfolio

A responsive, modern personal portfolio website built to showcase my backend-focused full stack engineering experience, projects, and skills.

## 🚀 Technologies Used
- **Framework**: [React](https://react.org/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Motion (Framer Motion)](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🌟 Features
- Fully responsive design (Mobile, Tablet, Desktop)
- Dark / Light mode toggle
- Integrated Resume Download
- Interactive UI components and smooth scroll navigation
- Clean, modern design layout

## 🛠️ Local Development

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd <your-repo-directory>
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. Open the localhost URL provided in your terminal in your browser.

## 📦 Deployment
This project is a static site (SPA) and can be easily hosted for free on platforms like:
- [Vercel](https://vercel.com/docs/frameworks/vite)
- [Netlify](https://docs.netlify.com/integrations/frameworks/vite/)
- [GitHub Pages](https://docs.github.com/en/pages)
- [Cloudflare Pages](https://developers.cloudflare.com/pages/)

To build the project for production, run:
```bash
npm run build
```
This will generate a `dist/` folder containing the optimized static assets that you can manually deploy or connect via Git.

## 📄 Resume Setup
Your resume is configured to be downloaded from the public assets. Make sure your `resume.pdf` file is placed directly inside the `public/` directory so it can be served correctly.
